# NotebookLM--Autismo-e-o-mercado-de-trabalho
Projeto sobre Autismo e Mercado de Trabalho desenvolvido com apoio do NotebookLM. Reúne insights sobre inclusão, desafios, habilidades e boas práticas para empregabilidade de pessoas no espectro autista, promovendo ambientes mais acessíveis e o uso de IA para organizar e analisar conhecimento.
